# LinkedIn Data Extractor

A production-ready Python tool for extracting structured data from public LinkedIn profiles using Selenium and BeautifulSoup.

---

## 🚀 Features

- Extracts full name, headline, location, experience, education, and skills
- Handles pagination and dynamic content
- Exports data to CSV and JSON
- Built-in rate limiting and error handling
- Supports batch processing from a list of URLs
- Lightweight and easy to extend

---

## 🛠️ Tech Stack

- Python 3.9+
- Selenium
- BeautifulSoup4
- Pandas
- WebDriver Manager
